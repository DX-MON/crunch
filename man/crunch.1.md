# NAME

**crunch** - C test harness and execution engine

# SYNOPSIS

| **crunch** \[**-h**|**\--help**]
| **crunch** \[**-v**|**\--version**]
| **crunch** \[**\--log** _file_] _TESTS_

# DESCRIPTION

# OPTIONS

-v, \--version

:   Prints the version information for crunch

-h, \--help

:   Prints this help message

\--log

:   Tells the engine to log all test output to the file named
